##正则表达式？

正则表达式（Regular-Expression）是使用单个字符串来描述、匹配一系列符合某个句法规则的字符串。简言之，就是按照某种规则匹配符合条件的字符串。

##RegExp作用

用一种简单规则处理复杂字符串

##RegExp对象

javascript通过内置对象RegExp支持正则表达式，并且有两种方法实例化RegExp对象：
（1）字面量:

var reg = /\bis/b/g;
'jack is a handsome boy'.replace(reg,'lesin');

（2）构造函数:

var reg = new RegExp('\\bis\\b','g');
//js中反斜线\是特殊字符，使用需要转义\\。

##修饰符

1、g:global 全文搜索，不写g时，搜索到第一个匹配内容就停止
2、i:ignore case 忽略大小写，默认是不忽略
3、m:multiple lines 多行搜索
