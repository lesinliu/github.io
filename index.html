JS 介绍

1.JS是什么–

Javascript是一种直译式脚本语言，是一种动态类型、弱类型、基于原型的解释语言，内置支持类型。也是一种面向对象语言。事件驱动。
JS难点主要在：语法的松散、浏览器兼容

  • 早期是为了做数据验证
  • 将动态数据写入HTML（主要用ajax）
  • 做动画特效
  • 操作DOM读写HTML
  • ……

2.JS组成–

  • ECMAscript（语法规范）
    数据类型、运算符、流程控制、函数、面向对象。。。
  • DOM（document object model）
  • BOM（browser object model）

3.JS的引入方式（3.4.均不常用）

  1. 内嵌
    <script></script>
  2. 外链
    <script src="#"></script>
  3. JS重定向
    <a href="javascript:#">haha</a>
    还有表单重定向，表单数据先用JS做验证，符合要求在提交服务器
  4. 在事件后面调用

4.JS的输出工具

  1. alert（””）
  2. document.write(“”)
  3. prompt(“”) 返回一个字符串类型

5.JS注释

  1. // 单行注释
  2. /*注释内容*/ 多行注释

6.javascript变量

可以把变量当做一个存储数据的容器。它可以存储任何数据。
用关键字var来创建变量

  1. 先声明后赋值
    var a; a=11;
  2. 声明并赋值
    var b = "love";
  3. 声明多个变量，在赋值
    var a,b; a=1; b=2;
  4. 声明多个变量并赋值
    var name="lesin",age=25;

7.javascript命名规范

  1. 不能以关键字或者保留字命名
  2. 必须以字母或者_或者$开头
  3. 驼峰命名法（或者下划线隔开 ）
  4. 严格区分大小写，并且命名要有意义
  5. 分号表示一段功能体的结束

##初识-统一资源定位符（URLs）

###统一资源定位符是什么

URL(Uniform Resource Location)汉译为：统一资源定位符。就是我们通俗讲的网址，URL可以由单词组成，例如：www.google.com,或者是因特网协议（IP）地址：xxx.xxx.xxx.xx等一串数字组成,名称比数字好记，所以我们在网上冲浪的时候大都键入域名。

###统一资源定位符的作用

URL是对可以从互联网上得到的资源的位置和访问方法的一种简洁的表示，是互联网上标准资源的地址。并且互联网上每一个文件的URL都是唯一的。最初是由蒂姆-伯纳斯-李发明用来作为万维网的地址，现在已经被万维网联盟编制为互联网标准RFC1738了。

###统一资源定位符的语法

HTTP的URL一般形式是：

'scheme://host.domain:port/path/filename'

scheme - 定义因特网服务的类型。最常见的类型是
http:超文本传输协议；普通网页，默认端口80
https:安全超文本传输协议；安全网页，会加密所有信息交换。默认端口443
ftp:文件传输协议；文件的上传或者下载。
host - 定义域主机（http 的默认主机是 www）
domain - 定义因特网域名，比如 w3school.com.cn
:port - 定义主机上的端口号（http 的默认端口号是 80，通常会被省略）
path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
filename - 定义文档/资源的名称

##正则表达式？

正则表达式（Regular-Expression）是使用单个字符串来描述、匹配一系列符合某个句法规则的字符串。简言之，就是按照某种规则匹配符合条件的字符串。

##RegExp作用

用一种简单规则处理复杂字符串

##RegExp对象

javascript通过内置对象RegExp支持正则表达式，并且有两种方法实例化RegExp对象：
（1）字面量:

var reg = /\bis/b/g;
'jack is a handsome boy'.replace(reg,'lesin');

（2）构造函数:

var reg = new RegExp('\\bis\\b','g');
//js中反斜线\是特殊字符，使用需要转义\\。

##修饰符

1、g:global 全文搜索，不写g时，搜索到第一个匹配内容就停止
2、i:ignore case 忽略大小写，默认是不忽略
3、m:multiple lines 多行搜索
