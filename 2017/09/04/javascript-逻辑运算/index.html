摘要

1、 逻辑运算符用于原始Boolean值运算，返回一个Boolean值，此时的运算就按照常用真值表理解。
2、 当运算数不是原始的Boolean类型，逻辑运算或将返回一个非布尔值。

逻辑运算符






















运算符 例子 说明
逻辑与（&&） eg1 && eg2 如果eg1能转换为false则返回eg1，否则返回eg2.同理在俩都是Boolean值时，
两个操作数都是true才返回true,否则返回false.
逻辑或（||） eg1 || eg2 如果eg1能转化为true，则返回eg1，否则返回eg2.同理在Boolean环境（或if条件判断中），两个操作数结果只要一个为true，返回true；都为false才返回false
逻辑非（！） !eg1 如果单个表达式能转化为true，则返回false，否则返回true

能被转换为false的表达式


  • null

  • NaN

  • 0

  • 空字符串“ ”

  • undefined
