JS 介绍

1.JS是什么–

Javascript是一种直译式脚本语言，是一种动态类型、弱类型、基于原型的解释语言，内置支持类型。也是一种面向对象语言。事件驱动。
JS难点主要在：语法的松散、浏览器兼容

  • 早期是为了做数据验证
  • 将动态数据写入HTML（主要用ajax）
  • 做动画特效
  • 操作DOM读写HTML
  • ……

2.JS组成–

  • ECMAscript（语法规范）
    数据类型、运算符、流程控制、函数、面向对象。。。
  • DOM（document object model）
  • BOM（browser object model）

3.JS的引入方式（3.4.均不常用）

  1. 内嵌
    <script></script>
  2. 外链
    <script src="#"></script>
  3. JS重定向
    <a href="javascript:#">haha</a>
    还有表单重定向，表单数据先用JS做验证，符合要求在提交服务器
  4. 在事件后面调用

4.JS的输出工具

  1. alert（””）
  2. document.write(“”)
  3. prompt(“”) 返回一个字符串类型

5.JS注释

  1. // 单行注释
  2. /*注释内容*/ 多行注释

6.javascript变量

可以把变量当做一个存储数据的容器。它可以存储任何数据。
用关键字var来创建变量

  1. 先声明后赋值
    var a; a=11;
  2. 声明并赋值
    var b = "love";
  3. 声明多个变量，在赋值
    var a,b; a=1; b=2;
  4. 声明多个变量并赋值
    var name="lesin",age=25;

7.javascript命名规范

  1. 不能以关键字或者保留字命名
  2. 必须以字母或者_或者$开头
  3. 驼峰命名法（或者下划线隔开 ）
  4. 严格区分大小写，并且命名要有意义
  5. 分号表示一段功能体的结束

JS的数据类型

1. 基本类型

  • undefined
    eg:var hd;
    声明变量但是不赋值，即为undefined

  • Null
    空对象

  • Number
    浮点数（小数）和整数都是数值类型

  • String

  • Boolean 布尔值
    只有真、假两个值

2. 引用类型

  • object

JS的运算符(JS运算能力一般，一般不用来进行精准的计算)

  • 算术运算符
    +、-、*、/、%、++、--
    在算术运算中，++a（自私运算）,表示先自增，在运算；a++（无私运算）表示，先参与运算，在自增。
  • 关系运算符（比较运算符）
  • 赋值运算符

  • 逻辑运算符

    逻辑运算符在对任意数据运算时，会将其转化为布尔值
    1.undefined false
    2.Null false
    3.Boolean 就是本身值
    4.Number 除了0以外都是true
    5.String 除了空字符串以外都是真
    6.对象 真

  • 三元运算符

    (条件)？条件为真的返回值：条件为假的返回值；
