hexo+github博客搭建详细教程

发表于 2017-08-19 |
字数统计 856 | 阅读时长 3

为了搭建一个独立博客，也是磕磕绊绊用了不少方法，大学时拿着学生优惠买几年阿里云，大材小用的做好了网站、博客，后期却总是荒废。最后选定了hexo和github来做个人博客，简单方便。。。

教程开始

1、环境配置、本地博客搭建

（1）安装node.js
（2）安装git
（3）安装hexo

node.js安装

点击node.js，选择适合你电脑系统的node.js下载按照默认配置安装即可。安装完成后在命令行工具输入node -v
查看是否安装成功，并输入npm -v检查npm是否安装。出现以下内容即表示成功安装

git安装

同样点击 git，选择适合自己操作系统的版本下载安装。

hexo安装

使用npm安装Hexo：
右键桌面任意空白处选择git Base Here
输入：
npm install -g hexo-cli

初始化博客根目录：
在电脑合适位置新建一个名为myblog（可自定）的文件夹，并在该文件夹下右键选择git Base Here输入hexo init myblog

到此我们的本地化博客已搭建完成，在myblog文件夹下右键选择git Base Here，输入命令hexo g回车待运行结束后在输入hexo s开启本地服务器，打开浏览器键入localhost:4000即可预览博客。如下图：


配置git：
将Git与GitHub帐号绑定：

电脑桌面任意位置右键选择输入
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"
生成ssh密钥：
ssh-keygen -t rsa -C "你的GitHub注册邮箱"
此处输入直接键入三个回车即可，不必输入密码。

4、绑定域名

（1）、设置CNAME
在建立github仓库是我们在仓库–setting–GitHub Pages下已经设置了域名，所以我们只需要在博客根目录–source文件夹下创建一个CNAME文件（只是一个CNAME文件，没有后缀）并在文件中写入我们的域名
